# Geo-Zones

**Note**: When choosing geo IDs for private testnet, consider using main region IDs like "0A00", "0B00", "0C00", "0D00" or "0E00".

- **Global**: "0001"

- **North America**: "0A00"
  - USA East: "0A01"
  - USA West: "0A02"
  - Canada: "0A03"
  - Mexico: "0A04"
- **South America**: "0B00"
  - SA East: "0B01"
  - SA West: "0B02"
- **Europe**: "0C00"
  - EU West: "0C01"
  - EU North: "0C02"
  - EU Central: "0C03"
- **Asia-Pacific**: "0D00"
  - AP Northeast: "0D01"
  - AP Southeast: "0D02"
  - AP South: "0D03"
  - Australia: "0D04"
- **Middle East & Africa**: "0E00"
  - Middle East: "0E01"
  - South Africa: "0E02"
  - North Africa: "0E03"
